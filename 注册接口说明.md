注册页面点击立即注册按钮之后需要向后端真正发送请求

请求示例代码如下

~~~javascript
var myHeaders = new Headers();
myHeaders.append("Content-Type", "application/json");

var raw = JSON.stringify({
   "username": "string",//就是表单中的nickname
   "pwd": "string", //password
   "phone": "string",
   "email": "string",
   "code": "string", //emailCode
   "token": "string" //从localStorage中的emailVerifyToken字段获取
});

var requestOptions = {
   method: 'POST',
   headers: myHeaders,
   body: raw,
   redirect: 'follow'
};

fetch("http://115.190.40.44:45444/user/register?username&pwd&phone&email&code", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
~~~

响应示例：

~~~json
{
    "success": true,
    "errCode": 0
}
~~~

响应成功之后跳转回登录页面
