在RepaymentRecords还款记录这个页面中需要真正向后端请求数据

从localStorage中获取accessToken并向后端发送请求

请求样例代码如下

~~~JavaScript
var myHeaders = new Headers();
myHeaders.append("Authorization", "");
myHeaders.append("Content-Type", "application/json");

var raw = JSON.stringify({});

var requestOptions = {
   method: 'GET',
   headers: myHeaders,
   body: raw,
   redirect: 'follow'
};

fetch("http://115.190.40.44:45444/loan/getRepayment", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
~~~
