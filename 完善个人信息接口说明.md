需要在profile页面添加一个个人资料的按钮，点击按钮之后进入一个新的页面，该页面需要用户填写个人资料，包括：

### 年龄

### 居住情况

前端问题：`您当前的居住情况是？`

推荐前端选项：

- `与家人同住/免费居住`
- `租房`
- `自有住房`

映射方式：

- `与家人同住/免费居住` -> `1`
- `租房` -> `2`
- `自有住房` -> `3`

### 工作年限 employment_duration

- 前端问题：`您当前工作已持续多久？`
- 推荐前端选项：
  - `待业/无固定工作`
  - `未满1年`
  - `1到3年`
  - `4到6年`
  - `7年以上`

映射方式：

- `待业/无固定工作` -> `1`
- `未满1年` -> `2`
- `1到3年` -> `3`
- `4到6年` -> `4`
- `7年以上` -> `5`

### 职业类型、职业层级

- 前端问题：`您目前的职业情况最接近以下哪一项？`
- 推荐前端选项：
  - `待业/无固定职业`
  - `非熟练劳动/基础岗位`
  - `熟练员工/普通职员`
  - `管理/自营/专业人员`

映射方式：

- `待业/无固定职业` -> `1`
- `非熟练劳动/基础岗位` -> `2`
- `熟练员工/普通职员` -> `3`
- `管理/自营/专业人员` -> `4`

### 家庭负担人数

- 推荐前端选项：
  - `0到2人`
  - `3人及以上`

映射方式：

- `0到2人` -> `2`
- `3人及以上` -> `1`

###  是否有房产/车辆等资产证明

- 前端问题：`您目前是否拥有以下可证明资产？`
- 推荐前端选项：
  - `无可证明资产`
  - `有车辆或一般资产`
  - `有储蓄/保险类资产证明`
  - `有房产证明`

映射方式：

- `无可证明资产` -> `1`
- `有车辆或一般资产` -> `2`
- `有储蓄/保险类资产证明` -> `3`
- `有房产证明` -> `4`

填好的表单需要发送如下请求，示例代码如下；

要在请求头中携带accessToken

~~~javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "");
myHeaders.append("Content-Type", "application/json");

var raw = JSON.stringify({
   "age": 0,
   "housing": 0,
   "employmentDuration": 0,
   "job": 0,
   "peopleLible": 0,
   "property": 0
});

var requestOptions = {
   method: 'POST',
   headers: myHeaders,
   body: raw,
   redirect: 'follow'
};

fetch("http://115.190.40.44:45444/user/riskProfile", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
~~~
