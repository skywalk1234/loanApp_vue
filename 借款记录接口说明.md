获取借款记录需要真正向后端请求数据，获取localStorage中的accessToken并请求， 请求示例代码如下：

~~~javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "");

var requestOptions = {
   method: 'GET',
   headers: myHeaders,
   redirect: 'follow'
};

fetch("http://115.190.40.44:45444/loan/orderList", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
~~~

在每一个对应的借款记录中都有一个按钮，展示贷款产品名称，点击这个按钮之后能知道这个贷款产品的详细信息。这个名称通过一下接口获得

请求示例

~~~javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "");
myHeaders.append("Content-Type", "application/json");

var raw = JSON.stringify({
   "id": 0  //这个id填的就是前面得到的product_id
});

var requestOptions = {
   method: 'POST',
   headers: myHeaders,
   body: raw,
   redirect: 'follow'
};

fetch("http://115.190.40.44:45444/loan/getProduct", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
~~~

响应示例：

