点击确认提前还款按钮之后，需要从localStorage中获取accessToken向后端发送请求

请求示例代码如下：

path指的是还款方式，根据用户选择减少期数还是减少月供，分别传递“TERM_REDUCTION”或者“PAYMENT_REDUCTION”

~~~javascript
var myHeaders = new Headers();
myHeaders.append("Authorization", "");
myHeaders.append("Content-Type", "application/json");

var raw = JSON.stringify({
   "loanId": 0,
   "amount": "string",
   "path": "string" 
});

var requestOptions = {
   method: 'POST',
   headers: myHeaders,
   body: raw,
   redirect: 'follow'
};

fetch("http://115.190.40.44:45444/loan/repay", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
~~~

拿到的响应的样例是这样的：

~~~json
{
    "success": true,
    "errCode": 0
}
~~~

如果成功就跳转到还款记录页面
